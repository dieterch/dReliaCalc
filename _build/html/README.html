dMyplant¶

Python Interface to INNIO MyPlant.

Easy access to INNIO Assets via Myplant API. Download Pandas Dataframes on a predefined Validation Fleet. Calculate demonstrated Reliability based on Success run Method and Lipson Equality approach as described in A.Kleyners Paper “Reliability Demonstration in Product Validation Testing”.

Installation¶

Windows, OS X & Linux:

pip install dMyPlant
conda install dMyPlant
